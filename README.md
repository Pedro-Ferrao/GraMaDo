# 🍽️ GraMaDo - Gastronomia Rústica

O **GraMaDo** é um projeto de landing page para um restaurante de gastronomia rústica contemporânea. O foco principal foi o desenvolvimento de uma interface imersiva, utilizando uma paleta de cores que remete aos elementos naturais (terra e fogo) e uma tipografia elegante para transmitir sofisticação.

🔗 **[Acesse o projeto online aqui](https://gramado.vercel.app/)**

---

## 🎯 Sobre o Projeto

Este site foi criado para demonstrar habilidades de **Front-end e UI Design**. A proposta é oferecer uma experiência visual que "venda" o conceito do restaurante antes mesmo do cliente chegar ao local, focando em conversão através de botões de reserva e visualização do cardápio.

### ✨ Funcionalidades
- **Layout Responsivo:** Experiência otimizada para Desktop, Tablet e Mobile.
- **Navegação Fluida:** Menu de navegação fixo e links internos.
- **Seção de Cardápio:** Estrutura clara para exibição de pratos e preços.
- **Identidade Visual:** Uso estratégico de contraste e tipografia serifada para um ar "premium".

---

## 🛠️ Tecnologias Utilizadas

Para garantir leveza e performance, o projeto foi desenvolvido com tecnologias nativas:

* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização, Flexbox, Grid e variáveis para cores.
* **JavaScript:** Interações de scroll e comportamento da interface.
* **Vercel:** Deployment e hospedagem.

---

## 🎨 Design System

* **Cores principais:** * `#2A211B` (Café Profundo)
    * `#C8845F` (Terracota / Fogo)
    * `#F5F5F5` (Off-white para contraste)
* **Tipografia:** Mix entre fontes clássicas (Serif) para títulos e modernas (Sans-serif) para leitura rápida.
